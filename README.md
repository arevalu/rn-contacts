# rn-contacts

React Native custom module to get contact list with Kotlin (Android) and Swift (iOS)

## Installation

```sh
npm install rn-contacts
```

## Usage

```js
import { multiply } from "rn-contacts";

// ...

const result = await multiply(3, 7);
```
